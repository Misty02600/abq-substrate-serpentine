## Task Creation Guide

使用以下命令管理任务：
- `add task` 或 `create task` - 创建新任务
- `update task [ID]` - 更新特定任务
- `show tasks [filter]` - 查看任务列表

有效的过滤器：
- `all` - 所有任务
- `active` - 进行中的任务
- `pending` - 待处理的任务
- `completed` - 已完成的任务
- `blocked` - 被阻塞的任务
- `recent` - 最近一周更新的任务
