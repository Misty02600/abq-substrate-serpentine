# Product Context

## Why This Project Exists
柔性电子器件中的蛇形导线设计是实现可拉伸电子产品的关键技术。本项目通过自动化有限元仿真，帮助研究人员快速评估不同设计参数对界面力学性能的影响。

## Problems It Solves

### 研究痛点
1. **建模效率低**：手动在 ABAQUS 中创建复杂的蛇形导线模型耗时费力
2. **参数扫描困难**：需要分析大量参数组合时，手动操作不可行
3. **结果提取繁琐**：从 ODB 文件中手动提取数据效率低下
4. **可重复性差**：手动建模容易出错，难以保证一致性

### 解决方案
- 参数化脚本自动生成模型
- 配置文件驱动的批量仿真
- 自动化后处理数据提取
- 标准化的模型命名规范

## How It Should Work

### 工作流程
```
config.ini → 解析配置 → 生成模型 → 划分网格 → 设置边界条件 → 提交作业 → 后处理
     ↓
多值参数展开 → 批量生成所有参数组合的模型
```

### 主要模块
1. **模型创建** (`build_solid_serpentine.py`)：创建实心基底模型
2. **配置解析** (`src/config/`)：处理 INI 配置文件
3. **几何建模** (`src/model/`)：基底、导线、孔隙的几何生成
4. **后处理** (`src/postprocess/`)：显示设置和数据提取
5. **工具函数** (`src/utils/`)：通用辅助功能

## User Experience Goals

### 使用便捷性
- 用户只需修改 `config.ini` 配置文件
- 一键运行即可生成所有模型
- 自动处理参数组合和命名

### 结果可追溯性
- 模型名称包含关键参数信息
- 配置文件可导出为 JSON 格式
- 完整的参数记录
